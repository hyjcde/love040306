# Love-0403 纪念网站开发计划

## 1. 项目背景与目标
基于之前旧版本的 `Love4U` 纪念网页（恋爱6个月纪念），我们计划使用现代化的前端技术栈（Next.js + React + Tailwind CSS）进行一次“系统升级”，为“0403”这个特殊的纪念日打造一个全新的、具有多端适配、丝滑交互体验的纪念网站。

## 2. 技术栈选择
- **框架**: Next.js (App Router) + React
- **样式**: Tailwind CSS (方便快速编写现代化的响应式页面)
- **开发语言**: TypeScript (增加代码健壮性)
- **部署**: 计划完成后使用 Vercel 免费部署上线。

## 3. 核心功能模块设计

1. **首页时间轴 (Hero Section)**
   - **核心功能**: 展示在一起的总天数（天/时/分/秒实时跳动计时器）。
   - **视觉设计**: 背景使用最喜欢的合照或轮播图，页面加载时自动播放背景音乐（旧项目的 `music.mp3`）。

2. **我们的故事 (Story Timeline)**
   - **核心功能**: 以时间轴形式记录从相识、相恋到现在的点点滴滴。
   - **视觉设计**: 垂直或水平滚动的卡片，附带动画效果。

3. **回忆画廊 (Photo Gallery)**
   - **核心功能**: 照片墙，展示去过的地方（如旧项目中的“外滩”、“黄浦江上”等合照）。
   - **视觉设计**: 采用现代瀑布流布局或支持手势滑动的 3D 轮播画廊。

4. **打字机与留言板 (Message Board)**
   - **核心功能**: 慢速打字机效果写下相对彼此说的话（复刻之前的 `typeWriter.js` 效果）。
   - **视觉设计**: 温馨、浪漫的字体和排版。
